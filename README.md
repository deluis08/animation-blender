# Animation Blender
A simple tool to automating the process of combining multiple FBX animations (for example from Mixamo) in one single .gltf or .glb file

**Requirements**
- blender
- python3

**Usage**
<pre><code>blender -b -P animation-blender.py -- rigged_character.fbx animation_folder ./output.(gltf | glb)</code></pre>
